Semantic Web Assignment 4 My Pizza Ontology

Welcome to my Semantic Web application! With the help of this application you can send a SPARQL query to my custom Stardog triple store containing the ontology from this link. It's a pizza ontology and the box below can be utilised to send your own query to the Stardog triple store. Furthermore you can click the buttons to check the difference between queries with inferencing and without inferencing! The output of the query will show up to the right.

Click the button below to GET the query to our triple store. You can either choose to run the query with or without inferencing. No worries, you can swap to and fro as much as you like.

  • without inferencing
  • with inferencing

The difference in results exists because with the use of inferencing the reasoner uses information already known from within the ontology to infer 'new' information. The query uses logic to infer information. This is why there are more results when applying inferencing. You can see for yourself using this application.
